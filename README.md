## 技术栈

## 页面

| 路由 | 说明 |
|------|------|
| `/` | Hero 首页 — 头像、社交链接、浮动科技图标、成功消息动画 |
| `/about` | 关于我 — 个人介绍、成长时间线、Bento Grid、技术栈、人生哲学、未来规划 |
